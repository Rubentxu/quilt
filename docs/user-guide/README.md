# Quilt User Guide

Welcome to Quilt, an AI-first knowledge graph for networked thought.

## Getting Started

### Creating Pages
1. Click "New Page" or use `Cmd/Ctrl+N`
2. Start typing to create blocks
3. Press Enter for new block, Tab to indent

### Navigation
- **Journal**: Daily notes with calendar
- **Pages**: All pages view
- **Search**: Full-text search across all content
- **Query**: SQL-like queries with the Query DSL
- **Graph**: Visual graph of page connections

### Block Operations
- **Enter**: New block below
- **Tab**: Indent (make child)
- **Shift+Tab**: Dedent (make sibling)
- **Backspace** at start: Merge with previous
- **/**: Open slash command palette

### References
- `[[Page Name]]`: Link to a page
- `((block-uuid))`: Reference to a specific block
