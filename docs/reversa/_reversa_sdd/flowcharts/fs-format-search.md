### MemoryFs - Algoritmo de mkdir-recur

```mermaid
flowchart TD
    Start(["<mkdir-recur!(dir)"])
    
    Init["fpath = url-to-path(dir)<br/>sub-dirs = p/loop"]
    
    Loop1{"exists? = <exists>(top-parent)"}
    CheckExists{exists?}
    
    AddRemain["remains = conj(top-parent)<br/>recur(parent, remains)"]
    DoneRemains["reverse remains<br/>→ sub-dirs"]
    
    Loop2{empty? sub-dirs}
    Mkdir1["mkdir(first)<br/>recur(rest)"]
    Done["resolved nil"]
    
    Start --> Init
    Init --> Loop1
    Loop1 --> CheckExists
    CheckExists -->|"yes"| DoneRemains
    CheckExists -->|"no"| AddRemain
    AddRemain --> Loop1
    
    DoneRemains --> Loop2
    Loop2 -->|"no"| Mkdir1
    Mkdir1 --> Loop2
    Loop2 -->|"yes"| Done
```
